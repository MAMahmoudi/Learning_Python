Exercises

    • Create variables for name, age, and GPA and print them.
    • Convert a float to an integer and observe what changes.
    • Swap the values of two variables without using a third variable.
    • Ask the user for a sentence and print its length.
    • Extract the first and last character of a string.
    • Count how many times a letter appears in a given word.
    • Check if a number is even or odd.
    • Determine if a year is a leap year.
    • Compare three numbers and print the largest.
    • Print numbers from 1 to 100 using a loop.
    • Sum all numbers from 1 to n.
    • Print a multiplication table for any number.
    • Find the maximum element in a list without using max().
    • Remove duplicates from a list.
    • Reverse a list manually using a loop.
    • Write a function that returns the factorial of a number.
    • Create a function that checks if a string is a palindrome.
    • Build a function that calculates BMI from weight and height.
    • Create a dictionary of student grades and compute the average.
    • Count word frequencies in a sentence.
    • Merge two dictionaries into one.
    • Simple calculator (add, subtract, multiply, divide).
    • Guess-the-number game using random numbers.
    • To-do list program with add/remove/display options.
