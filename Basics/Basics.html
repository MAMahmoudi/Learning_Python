Comments

Comments in Python are used to explain code and make it more readable. They are ignored by the interpreter during execution. Single-line comments start with the hash symbol (#), while multi-line comments can be created using triple quotes (''' or """). 
# This is a single-line comment
'''
This is a multi-line comment.
It can span multiple lines.
'''

Variables

Python variables are used to store data values.
Variable names must start with a letter or an underscore (_) and can contain letters, numbers, and underscores.
They are case-sensitive (e.g., myVar and myvar are different variables).
A variable is created by assigning a value to it using the equals sign (=).
For example: 
# Creating a variable
x = 5
y = "Hello, World!"
print(x)
print(y)
Python is dynamically typed, so you don't need to declare the variable type explicitly.
It infers the type based on the assigned value.

Data Types

Python has several built-in data types, including:
  • Integers (int): Whole numbers, e.g., 1, 2, 3
  • Floating-point numbers (float): Decimal numbers, e.g., 1.5, 2.75
  • Strings (str): Text enclosed in quotes, e.g., "Hello"
  • Booleans (bool): True or False values
  • Lists: Ordered collections of items, e.g., [1, 2, 3]
  • Dictionaries: Key-value pairs, e.g., {"name": "Alice", "age": 25}
# Examples of data types
integer_var = 10
float_var = 10.5
string_var = "Hello"
boolean_var = True
list_var = [1, 2, 3, 4, 5]
dict_var = {"name": "Alice", "age": 25}
print(type(integer_var))
print(type(float_var))
print(type(string_var))
print(type(boolean_var))
print(type(list_var))
print(type(dict_var))

Operators

Operators in Python are special symbols that perform operations on variables and values.
They can be categorized into several types:
  • Arithmetic Operators: +, -, *, /, %, **, //
  • Comparison Operators: ==, !=, >, <,>=, <=
  • Logical Operators: and, or, not
  • Assignment Operators: =, +=, -=, *=, /=
  • Membership Operators: in, not in
  • Identity Operators: is, is not
# Examples of operators
a = 10
b = 5
print(a + b)  # Addition
print(a - b)  # Subtraction
print(a * b)  # Multiplication
print(a / b)  # Division
print(a % b)  # Modulus (remainder)
print(a ** b) # Exponentiation
print(a // b) # Floor division
print(a == b) # Equal to
print(a != b) # Not equal to
print(a > b)  # Greater than
print(a < b)  # Less than
print(a >= b) # Greater than or equal to
print(a <= b) # Less than or equal to
print(a > 5 and b < 10)  # Logical AND
print(a > 5 or b > 10)   # Logical OR
print(not(a > b))  # Logical NOT
print(a in [5, 10, 15])  # Membership operator
print(a is b)  # Identity operator

Strings

Strings in Python are sequences of characters enclosed in single or double quotes. They can be manipulated using various methods and operations. 
# Creating and manipulating strings
my_string = "Salem, ESST!"
print(my_string)  # Print the string
print(len(my_string))  # Get the length of the string
print(my_string.upper())  # Convert to uppercase
print(my_string.lower())  # Convert to lowercase
print(my_string.replace("ESST", "Python"))  # Replace substring
To get all the string methods, you can use the following code: 
print(dir(my_string))

String Formatting

String formatting allows you to create strings with dynamic content.
Python provides several ways to format strings, including f-strings, the format() method, and the % operator. 
# Using f-strings for string formatting
name = "Alice"
age = 25
formatted_string = f"My name is {name} and I am {age} years old."
print(formatted_string)

# Using the format() method for string formatting
formatted_string2 = "My name is {} and I am {} years old.".format(name, age)
print(formatted_string2)
# You can also use positional or keyword arguments in the format() method
formatted_string2_pos = "My name is {0} and I am {1} years old.".format(name, age)
formatted_string2_kw = "My name is {name} and I am {age} years old.".format(name=name, age=age)
print(formatted_string2_pos)
print(formatted_string2_kw)

# Using the % operator for string formatting
formatted_string3 = "My name is %s and I am %d years old." % (name, age)
print(formatted_string3)
When using the % operator, %s is used for strings and %d is used for integers.

Conditional Statements

Conditional statements allow you to execute different blocks of code based on certain conditions. The primary conditional statements in Python are if, elif, and else. 
# Example of conditional statements
x = 10
if x > 0:
    print("x is positive")
elif x == 0:
    print("x is zero")
else:
    print("x is negative")
The pass statement is used as a placeholder when no action is required.
We can also use ... to indicate an incomplete block of code. 
# Using pass and ...
if True:
    pass  # Placeholder for future code
else:
    ...  # Placeholder for future code

Input

The input() function is used to take input from the user. It reads a line from input, converts it into a string, and returns it. 
# Taking input from the user
name = input("Enter your name: ")
age = input("Enter your age: ")
print(f"Hello, {name}! You are {age} years old.")
By default, the input() function returns the input as a string. If you need to convert it to another data type (e.g., int or float), you can use the appropriate type conversion functions. 
# Converting input to different data types
age = int(input("Enter your age: "))  # Convert input to integer
height = float(input("Enter your height in meters: "))  # Convert input to float
print(f"You are {age} years old and {height} meters tall.")

Lists

Lists in Python are ordered collections of items that can be of different data types. They are mutable, meaning you can change their content after creation. 
# Creating and manipulating lists
my_list = [1, 2, 3, 4, 5]
print(my_list)
my_list.append(6)  # Add an item to the end of the list
print(my_list)
my_list.remove(3)  # Remove an item from the list
print(my_list)
The elements of a list can be of different data types, including other lists. There are various methods available for list manipulation, such as insert(), pop(), sort(), and reverse(). 
# Example of a list with different data types
mixed_list = [1, "Hello", 3.14, [2, 4, 6]]
print(mixed_list)
# Accessing elements in a nested list
print(mixed_list[3])      # Output: [2, 4, 6]
print(mixed_list[3][1])   # Output: 4
# Slicing a list
sub_list = my_list[1:4]  # Get elements from index 1 to 3
print(sub_list)
# More list methods
my_list.insert(2, 10)  # Insert 10 at index 2
print(my_list)
popped_item = my_list.pop()  # Remove and return the last item
print(popped_item)
print(my_list)
my_list.sort()  # Sort the list
print(my_list)
my_list.reverse()  # Reverse the list
print(my_list)

Dictionaries

Dictionaries in Python are collections of key-value pairs. They are mutable and allow you to access values by their keys.
Keys must be unique and immutable (e.g., strings, numbers, or tuples), while values can be of any data type. They must be hashable. Meaning, they must have a hash value that remains constant during their lifetime. Data types like lists and dictionaries are not hashable and cannot be used as keys. 
# Creating and manipulating dictionaries
my_dict = {"name": "Alice", "age": 25}
print(my_dict)
my_dict["city"] = "New York"  # Add a new key-value pair
print(my_dict)
del my_dict["age"]  # Remove a key-value pair
print(my_dict)
# Accessing values by keys
print(my_dict["name"])
# Getting all keys and values
print(my_dict.keys())
print(my_dict.values())
# Pop method to remove a key and return its value
age = my_dict.pop("age", "Not Found")  # Returns "Not Found" if key doesn't exist
print(age)
print(my_dict)
Iterating through a dictionary can be done using a for loop to access keys, values, or key-value pairs. 
# Iterating through a dictionary
for key in my_dict:
    print(f"Key: {key}, Value: {my_dict[key]}")
# Or using items() to get key-value pairs
for key, value in my_dict.items():
    print(f"Key: {key}, Value: {value}")

Sets

Sets in Python are unordered collections of unique items. They are mutable and support mathematical set operations like union, intersection, and difference. 
# Creating and manipulating sets
my_set = {1, 2, 3, 4, 5}
print(my_set)
my_set.add(6)  # Add an item to the set
print(my_set)
my_set.remove(3)  # Remove an item from the set
print(my_set)
# using pop
popped_item = my_set.pop()  # Remove and return an arbitrary item
print(popped_item)
print(my_set)
# Set operations
set_a = {1, 2, 3}
set_b = {3, 4, 5}
print(set_a.union(set_b))        # Union {1, 2, 3, 4, 5}
print(set_a.intersection(set_b)) # Intersection {3}
print(set_a.difference(set_b))   # Difference {1, 2}
To access set elements, you can iterate through the set using a for loop since sets do not support indexing. 
# Iterating through a set
for item in my_set:
    print(item)
We can convert a set to a list as follows: 
# Converting a set to a list
set_as_list = list(my_set)
print(set_as_list)
# Convert a list to a set
list_as_set = set(set_as_list)
print(list_as_set)
Sets are mainly used for membership testing and eliminating duplicate entries from a collection.

Tuples

Tuples in Python are ordered collections of items that are immutable, meaning their content cannot be changed after creation. They are mainly used to group related data together. For example, you can use a tuple to represent a coordinate point (x, y). 
# Creating and manipulating tuples
my_tuple = (1, 2, 3, 4, 5)
print(my_tuple)
# Tuples are immutable, so you cannot add or remove items
print(my_tuple[0])  # Accessing elements by index
# Slicing a tuple
sub_tuple = my_tuple[1:4]  # Get elements from index 1 to 3
print(sub_tuple)
# Tuples can contain different data types
mixed_tuple = (1, "Hello", 3.14, (2, 4, 6))
print(mixed_tuple)

Loops

Loops in Python are used to execute a block of code multiple times. The two main types of loops are for loops and while loops.
A for loop iterates over a sequence (like a list, tuple, or string) or other iterable objects.
To break out of a loop, you can use the break statement.
To skip the current iteration and move to the next one, you can use the continue statement. 
# Example of a for loop
for i in range(5):
    print(i)
# Example of a while loop
count = 0
while count < 5:
    print(count)
    count += 1
# Using break and continue
for i in range(10):
    if i == 5:
        break  # Exit the loop when i is 5
    if i % 2 == 0:
        continue  # Skip even numbers
The range() function generates a sequence of numbers, which is commonly used in for loops.
The syntaxe of range() can be:
  • range(stop): Generates numbers from 0 to stop-1
  • range(start, stop): Generates numbers from start to stop-1
  • range(start, stop, step): Generates numbers from start to stop-1 with a specified step 
# Examples of range() function
print(list(range(5)))          # Output: [0, 1, 2, 3, 4]
print(list(range(2, 6)))       # Output: [2, 3, 4, 5]
print(list(range(1, 10, 2)))   # Output: [1, 3, 5, 7, 9]

List Comprehensions

List comprehensions provide a concise way to create lists in Python. They consist of brackets containing an expression followed by a for clause, and can include optional if clauses. 
# Example of a list comprehension
squares = [x**2 for x in range(10) if x % 2 == 0]
print(squares)
We can use the same method with sets and dictionaries: 
# Set comprehension
even_squares_set = {x**2 for x in range(10) if x % 2 == 0}
print(even_squares_set)
# Dictionary comprehension
squares_dict = {x: x**2 for x in range(10) if x % 2 == 0}
print(squares_dict)

Functions

Functions in Python are blocks of reusable code that perform a specific task. They are defined using the def keyword, followed by the function name and parentheses. 
# Defining and calling a function
def greet(name):
    return f"Hello, {name}!"
print(greet("Alice"))

Lambda Functions

Lambda functions are small anonymous functions defined using the lambda keyword. They can take any number of arguments but can only have a single expression.
The syntax is:
Func_Name = lambda arguments: expression 
# Example of a lambda function
add = lambda x, y: x + y
print(add(5, 3))
# Using lambda with map() function
numbers = [1, 2, 3, 4, 5]
squared_numbers = list(map(lambda x: x**2, numbers))
print(squared_numbers)
# Using lambda with filter() function
even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)

Modules

Modules in Python are files containing Python code that can be imported and used in other Python programs. They help organize code and promote reusability.
You can create your own modules or use built-in and third-party modules. Let's see how to create and use a simple module. 
# Creating a module named my_module.py
def add(a, b):
    return a + b
def subtract(a, b):
    return a - b
            
# Using the module in another Python file
import my_module
print(my_module.add(5, 3))
print(my_module.subtract(5, 3))
# Importing specific functions from the module
from my_module import add, subtract # or from my_module import * to import all functions
print(subtract(10, 4))
print(add(10, 4))

Classes and Objects

Classes in Python are blueprints for creating objects. They encapsulate data and behavior related to that data. An object is an instance of a class.
Classes are defined using the class keyword, followed by the class name and a colon.
The __init__() method is a special method used to initialize the attributes of the class.
Methods are functions defined within a class that operate on the object's data.
The self parameter refers to the instance of the class and is used to access its attributes and methods. 
# Defining a class and creating an object
class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    def introduce(self):
        return f"My name is {self.name} and I am {self.age} years old."
student1 = Student("Alice", 25)
print(student1.introduce())
There are other special methods in Python classes, such as __str__() for string representation and __repr__() for a detailed string representation. To see the list of all special methods, you can visit Python's official documentation. 
# Example of __str__ and __repr__ methods
class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    def __str__(self):
        return f"Student(name={self.name}, age={self.age})"
    def __repr__(self):
        return f"Student('{self.name}', {self.age})"
student1 = Student("Alice", 25)
print(student1)
print(repr(student1))
Inheritance allows a class (child class) to inherit attributes and methods from another class (parent class). This promotes code reusability and establishes a hierarchical relationship between classes. 
# Example of inheritance
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    def introduce(self):
        return f"My name is {self.name} and I am {self.age} years old."
class Student(Person):
    def __init__(self, name, age, grade):
        super().__init__(name, age)
        self.grade = grade
    def introduce(self):
        return f"My name is {self.name}, I am {self.age} years old, and my grade is {self.grade}."
student1 = Student("Alice", 25, "A")
print(student1.introduce())
In this example, the Student class inherits from the Person class, allowing it to use the introduce() method while adding its own attribute (grade) and overriding the introduce() method.

Pythonic Thinking

Pythonic thinking refers to writing code that follows the conventions and idioms of the Python programming language. It emphasizes readability, simplicity, and the use of Python's built-in features. Follow the PEP 8 Style Guide for Python Code to write Pythonic code.

Whitespace

  • Use spaces instead of tabs for indentation.
  • Use four spaces for each level of syntactically significant indenting.
  • Lines should be 79 characters in length or less.
  • Continuations of long expressions onto additional lines should be indented by four extra spaces from their normal indentation level. In a file, functions and classes should be separated by two blank lines.
  • In a class, methods should be separated by one blank line.
  • In a dictionary, put no whitespace between each key and colon, and put a single space before the corresponding value if it fits on the same line.
  • Put one—and only one—space before and after the = operator in a variable assignment.
  • For type annotations, ensure that there is no separation between the variable name and the colon, and use a space before the type information.
For example: 
# Correct indentation and spacing
def my_function(param1: int, param2: str) -> None:
    my_dict = {"key1": "value1", "key2": "value2"}
    total = param1 + 10
    print(total)

Naming

  • Functions, variables, and attributes should be in lowercase_underscore format.
  • Protected instance attributes should be in _leading_underscore format.
  • Private instance attributes should be in __double_leading_ underscore format.
  • Classes (including exceptions) should be in CapitalizedWord format.
  • Module-level constants should be in ALL_CAPS format.
  • Instance methods in classes should use self, which refers to the object, as the name of the first parameter.
  • Class methods should use cls, which refers to the class, as the name of the first parameter.
For example: 
# Correct naming conventions
class MyClass:
    def __init__(self):
        self.public_attr = "public"
        self._protected_attr = "protected"
        self.__private_attr = "private"

    def public_method(self):
        pass

    def _protected_method(self):
        pass

    @classmethod
    def class_method(cls):
        pass
    @staticmethod
    def static_method():
        pass
MY_CONSTANT = 42

Imports

  • Always put import statements (including from x import y) at the top of a file.
  • Always use absolute names for modules when importing them, not names relative to the current module's own path. For example, to import the foo module from within the bar package, you should use from bar import foo, not just import foo.
  • If you must do relative imports, use the explicit syntax from . import foo.
  • Imports should be in sections in the following order: standard library modules, third-party modules, your own modules. Each subsection should have imports in alphabetical order.
For example: 
# Correct import statements
from pandas import DataFrame
# Relative import
from . import my_module
