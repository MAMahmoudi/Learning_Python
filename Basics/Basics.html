Basics of Python

Compile and run a Python program

To compile and run a Python program, you can use the Python interpreter. Save your code in a file with a .py extension and run it using the command: python filename.py

Variables and Data Types

In Python, variables are used to store data. Common data types include integers, floats, strings, and booleans.

# Example of variables and data types
age = 25                      # Integer
name = "John"                 # String
height = 5.9                  # Float
is_student = True             # Boolean

Control Flow

Control flow statements like if, for, and while are used to control the execution of code based on conditions.

Functions

Functions are defined using the def keyword and are used to encapsulate reusable code blocks.

Modules and Packages

Modules are files containing Python code that can be imported into other Python files. Packages are collections of modules.
